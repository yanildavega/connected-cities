# Connected Cities

REST service that calculates if two cities are connected by roads. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

- JDK 8
- Maven 3.3+


### Installing

Download the source code in your local machine

~~~
git clone https://github.com/yanildavega/connected-cities.git
~~~

package the files

~~~
mvn package
~~~



## Deployment

After packaging the project through maven, deployment can be performed in any machine with Java 8 installed.

java -jar target/
